整合包说明

整合包介绍

此整合包是为BECUY服务器的游玩方便而开发的整合包，集合了多个模组，其中包括了核心模组MTR我的世界铁路 和 机械动力 优化了游戏性能，并提供了独特的游戏玩法。整合包专注于冒险、科技的平衡结合， 为玩家提供沉浸式的游戏体验。

主要特点

• 优化的游戏性能：即使在低配电脑上也能流畅运行
• 独特的任务系统：引导玩家体验整合包的全部内容
• 平衡的游戏机制：精心调整模组间的平衡性
• 自动化支持：丰富的科技模组满足自动化需求

系统要求

最低配置：
• 操作系统: Windows 10 64位
• 处理器: Intel i5 或同等性能处理器
• 内存: 8GB RAM
• 显卡: Intel IRIS xe GRAPHICS

推荐配置：
• 操作系统: Windows 10/11 64位
• 处理器: Intel i7 或同等性能处理器
• 内存: 8GB RAM
• 显卡: 任意显卡即可

整合包按键说明（公测后改动）

核心快捷键

  • M 键
    打开小地图设置界面，可调整地图大小、位置和显示内容
  • J 键
    打开任务系统界面，查看当前任务进度和奖励
  • U 键
    打开游戏内百科，查询模组物品的合成方式和用途
  • R 键
    查看物品合成表（将鼠标悬停在物品上按R键）

工具类快捷键

  • G 键
    打开/关闭游戏设置菜单
  • V 键
    打开物品管理器，搜索和定位背包中的物品
  • X 键
    打开经验管理界面，将经验储存到经验之书中
  • C 键
    打开技能树界面，查看和升级角色技能

其他重要按键

  • F7
    显示光照等级，红色表示怪物可生成区域
  • F9
    切换小地图类型（圆形、方形、缩略图）
  • Ctrl + O
    打开游戏优化设置，调整内存分配等高级选项
  • Alt + F
    打开快速建造菜单，选择预设建筑结构
